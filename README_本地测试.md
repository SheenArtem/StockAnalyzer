1. **打开命令行**
   ```powershell
   cd C:\GIT\StockAnalyzer
   ```

2. **安装依赖**（首次运行）
   ```powershell
   pip install -r requirements.txt
   ```

3. **启动应用**
   ```powershell
   streamlit run app.py
   ```

## 🎯 测试要点

### 1. 籌碼分布图表
进入「籌碼分析」标签，检查：

**✅ 颜色清晰**
- 外资：橙色
- 投信：红色
- 自营商：**淺綠色**（新改进）

**✅ 间隔明显**
- 柱状图之间有间隙
- 显示淡灰色网格线
- 零线加粗突出

### 2. 当冲周转概况
**✅ 数据合理**
- 当冲成交量：千位数（例如：6,281 张）
- 当日总量：万位数（例如：34,688 张）
- ❌ 不应该是百万级数字

### 3. 法人买卖超
**✅ 智能判断**
- 显示资金金额（百万）
- 或显示成交量占比
- 根据股价高低动态调整

## 🔧 常见问题

### Q: 端口被占用怎么办？
```powershell
# 更换端口
streamlit run app.py --server.port 8502
```

### Q: 数据加载很慢？
- 首次运行需要下载数据（30秒左右）
- 之后会使用缓存，很快（1-2秒）

### Q: 缺少某个模块？
```powershell
# 重新安装依赖
pip install -r requirements.txt --force-reinstall
```
