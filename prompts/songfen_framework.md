## 一、看公司：re-rate 訊號清單

市場不反應基本面本身，而反應**確定性的變化**。股票不會因為你研究對公司就上漲——要等市場願意重新定義這家公司。一次完整 re-rate 需要**現金流結構改變**才算數，敘事變了不保證發生。

### re-rate 五訊號（優先順序）

1. **收入變得更可預測**（從一次性變訂閱、從消費電子變 infrastructure）
2. **現金流更穩定**（市場為「可預測性」付錢，不是為「成長」）
3. **公司變成產業基礎設施**（整個產業都需要你的產品）
4. **產業競爭減少**（從多家廝殺變寡佔）
5. **管理層資本配置被信任**（不亂擴產、會回購、控費用）

### 結構性 re-rate 三條件（必須同時成立）

- 改變長期供給結構（去中東化 / 去單一依賴）
- 持續 CAPEX（公司自己相信是長期）
- 可持續現金流（長約、轉換成本、訂閱）

三個沒同時出現 → 只是政策 + 情緒，不是 re-rate。

### re-rate 三等級

- **假反彈**：只修復估值（例：1990 海灣戰爭後航空股）
- **節奏型**：變數排序看錯被修正（例：2022 航空股）
- **結構性**：重寫估值框架，估值地板永久抬升（例：1973 能源、2010s 雲端、AI infrastructure）

### 市場重新定價四階段

機構研究 → sellside 上調預期 → 媒體社群擴散 → ETF 被動資金。第四階段發生時早期資金已減碼。

### 定價三階段識別（用「好消息股價反應」判斷）

- 好消息**大漲** → 第一階段（估值擴張）
- 好消息**不動** → 第二階段（基本面追上估值）
- 好消息**下跌** → 第三階段（估值收縮）

偉大公司常經歷漫長第二階段（MSFT / AMZN / NFLX 都盤整多年後十倍）。真能穿越牛熊、反覆回第一階段的公司特徵是「敘事可循環」——MSFT（PC → Office → Azure → AI）、AMZN（電商 → AWS → AI+衛星）、COST（會員制現金流）。

## 二、看財報：5-layer 損益表拆解

**ROIC 比 EPS 重要。** EPS 告訴你「這季賺多少」，ROIC = NOPAT / Invested Capital 告訴你「每投 1 塊錢滾回多少」。Amazon 被市場誤會多年就是因為故意把護城河（物流、Prime 補貼、AWS 初期建設）藏在當期費用裡，EPS 醜但等規模效應出來時 ROIC 跳升，才 re-rate。

**真正爆發條件**：ROIC 上升 + Organic Growth 持續（同時發生）。
- 只有 ROIC 沒成長 → 成熟企業，估值穩定不 re-rate
- 只有成長沒 ROIC → 資本黑洞，最後會崩

**CCC（現金轉換週期）比 EPS 和 FCF 更早反映體質改善**——庫存快速消化、應收回收變快、現金回流，這些是「過程」，比結果早一季看到轉折。

### Layer 1：營收 — 「誰在成長」分三級

- **🥇 Recurring SaaS**（例：AWS backlog）— 市場最愛，拉估值
- **🥈 結構性增量**（例：AI 基建、新產品線）— re-rate 引擎
- **🥉 週期性 / 一次性**（例：舊硬體）— 不會給高倍數

反例：營收 +20% 全部來自低毛利硬體 → 虛胖成長，不 re-rate。

**進階：跳 BS 看兩個「未來營收蓄水池」**
- 合約負債（Deferred Revenue）持續增加 → 下一季營收已在排隊
- 預付帳款（Prepaid Expenses）暴增 → 公司看到後面訂單多到先砸錢鎖產能

### Layer 2：毛利 — 看邊際改善而非水準

**健康毛利上升（會 re-rate）**
- Product mix 改變：SaaS / AI 佔比上升
- 規模經濟：單位成本下降（議價力強）

**危險毛利下降（會 de-rate）**
- 降價搶市：營收 ↑ 但毛利 ↓（割肉換營收 = 失血）
- 庫存減損：毛利掉 5% 原因是提列損失（需求錯判 / 產品過時）

重點：**毛利不是結果，是「競爭力的證據」**。毛利從 15% 爬到 18% 的公司，推力遠大於長期 50% 毛利的穩定公司——3% 邊際改善代表「質變」。

### Layer 3：費用 — 分好費用 vs 壞費用

**好費用（投資，會 re-rate 前兆）**
- R&D ↑（開發新產品）
- SG&A ↑（擴張市場）
- 同時：新業務佔比 ↑、毛利不崩

**壞費用（燒錢，會 de-rate）**
- R&D ↑ 但新產品一直遞延
- 行政費 ↑（豪華辦公室、高層獎金）
- 沒帶動營收、毛利 ↓

**SBC 陷阱（股票薪酬）**：GAAP 列為費用讓淨利難看，Non-GAAP 會加回。分析師不買單因為「稀釋股權」。檢查法：股數每年 +5% 但營收只 +10% → 股東權益被稀釋。

### Layer 4：營業利益 — 營業槓桿是 re-rate 終極爆點

**營業槓桿白話**：麵館例子。店租固定，客人從 10 變 100，多收的錢扣食材幾乎全是純利。

**具體**：營收 +20%、費用（OpEx）因研發告一段落只 +5%，營業利益從 10M 暴增到 25M（+150%）。**這就是主升段。**

**反例**：營收 +20%、費用 +25%，利潤不動甚至下降 → 沒過臨界點或沒效率。

### Layer 5：EPS — 剝掉雜訊看「標準化獲利」

**常見 EPS 雜訊**
- 業外收益（賣地、處分轉投資）— 今年有明年沒
- 匯兌損益 — 純財務運氣
- 回購股票（分母變小）— 財務操作不是 re-rate
- 稅率下降（一次性退稅）— 看有效稅率，標準化重算

**「垃圾盈餘」特徵**：靠縮減研發（省錢）+ 賣資產（業外）+ 買回股票（分母變小）湊出來。這種成長撐不過兩季。

### 資本密集產業：PE 還是 PB？

- **用 PB**：產品同質化 + 供給可擴張 + 獲利暴起暴落（例：記憶體）
- **用 PE**：需求結構穩定 + 壟斷（例：台積電）
- **用 PE**：產品生命週期 10-15 年 + 客戶 design-in 切換成本（例：類比晶片）

資本黑洞不是 CAPEX 大不大，看「有沒有被鎖定的需求」「現金流是否更穩」「供給會不會失控」。

### 景氣循環產業的庫存

不看多寡，看三件事：變化方向、搭配營收、週轉速度。健康增加（上行期）vs 被動堆積（下行期）意義完全相反。

## 三、看買賣：擇時與紀律

### 停損 — 不是價格問題，是邏輯有沒有壞掉

機構只在 thesis 被推翻時停損。看錯分三種：
- **產業方向錯**（最致命）
- **公司競爭力看錯**
- **時間節奏看錯**（最常見——大部分人不是看錯方向而是撐不到）

**買之前就定義什麼叫看錯**：寫下為什麼買、預期會發生什麼、沒發生要怎麼做。沒有 exit 條件的投資最後都變成祈禱。

### 加碼 — 不看跌幅看風險訊號

永遠保留 dry powder。機構按訊號放錢而非按跌幅：

| 訊號 | 倉位 |
|------|------|
| 利空不再破底（CTA 賣壓鈍化）| 10-20%（測試單）|
| 強勢股撐住（資金沒撤出）| 30-50%（核心倉）|
| 市場分化（開始選股）| 60-80%（加碼）|
| 趨勢確認 | 接近滿倉 |

加碼要加**「最先不再跌的」**而不是「跌最多的」。止跌順序（權值股 → 成長股 → 高 Beta）= 加碼順序：權值止血、成長股賺錢、高 Beta 放大。

### 獲利了結 — 三段式節奏

1. 漲一段 → 賣 10-20%（降心理壓力）
2. 動能轉弱 → 賣 20-30%（保護成果）
3. 結構破壞 → 大幅減碼

賣出三大致命錯：
- 因為有賺錢而賣（用盈虧決定而非結構）
- 因為回檔而賣（用短期波動做反趨勢）
- 賣掉贏家留下輸家（組合越來越平均越沒動能）

### 反彈買點 — 交易「大家不再相信什麼」

市場只要不再變更壞就會反彈，不需要等利多。三風險指標只要**不再惡化**就夠：
- 信用利差（HY spread）
- VIX 期限結構（VIX3M - VIX）
- 十年期殖利率

用分段錯誤容忍而非 All-in。轉折一定在多數人還沒準備好時發生。

**沒部位**：不要一次補滿，等拉回或小部位重建節奏。
**部位不夠**：不要為了少賺冒本來不會冒的風險。
**賣飛**：當新標的重新評估，不要因為曾經擁有就追。

### 強勢股 — 下跌中被發現

上漲有很多原因（資金氾濫、被動推升、題材）但「不下跌」只有一種：有人在接。

**機構流程**：刻意看最醜幾天 → 建抗跌名單 → 反彈時看誰先創高。
**終極訊號**：對利空不反應（新聞壞股價不跌、開盤下殺被拉回、隔天創高）。

### 接受不完美

永遠不會買在最低也不會賣在最高。穩定的人不追求每次都做對，而是讓自己還在市場裡——資金、情緒、耐心三樣資產才是最重要的。

## 四、看市場：機構 vs 散戶常見錯誤

### 市場殘酷特性

- 當你感到安全時市場已定價完了
- 越確定價格越貴
- 報酬幾乎全來自「確定性提升的那一段」
- 多數人把「確定性」當「安全」→ 永遠最確定時追高、最不確定時停損 → 做「最情緒化時的決定」

### 對比表

| 面向 | 散戶 | 機構 |
|------|------|------|
| 決策依據 | 看價格 | 看風險 |
| 進場邏輯 | 越跌越想買、越漲越敢追 | 風險下降才進場 |
| 加碼方式 | 按跌幅機械分批 | 按訊號動態調整 |
| 遇虧損 | 強化信仰、找理由 | 修正假設、重寫 thesis |
| 看錯 | 情緒受不了停損 | 邏輯壞掉停損 |
| 盯的數字 | 股價、漲跌幅 | 信用利差、VIX、殖利率 |
| 交易頻率 | 越亂越想動作 | 沒有優勢時不交易 |
| 組合演化 | 賣強留弱 | 讓贏家跑、淘汰輸家 |
| 看新聞 | 追「已發生的事」 | 判斷「市場預期有沒有錯」 |

### 行為金融四陷阱

- **損失厭惡**讓你錯過第一段
- **近期偏誤**放大壞消息
- **後見之明**讓你追高
- **故事偏誤**讓你追新聞

這四件事連成循環 = 「永遠買在最舒服的地方」而那通常是風險報酬最差的地方。

### 散戶學投資正確順序

1. 會計財報（公司在賺什麼）
2. 產業（為什麼能賺）
3. 心理（為什麼會做錯）
4. 最後才是看盤

工具用來驗證想法，不是找標的。

### 判「相對低點」三訊號

- 利空不再破底
- 強勢股撐住
- 市場開始分化

**五個不出手訊號**：過度樂觀、消息太雜、基本面沒變但超跌、資金流向不明、指標混亂。

## 使用這份框架的正確方法

1. **不是 checklist**：不要把框架當打勾表，要當思考工具
2. **先問「現在市場在定什麼價」**，再問「我手上這支股票的敘事有沒有改變」
3. **永遠準備反面論點**——找「為什麼我會看錯」比找「為什麼我是對的」更有價值
