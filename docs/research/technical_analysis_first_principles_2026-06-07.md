# 用第一性原理拆解「技術分析」

> 撰於 2026-06-07。本文把技術分析剝到公理層重新推導，並以本專案累積的實證資料
> （`reports/indicator_ic_matrix.csv` / `reports/indicator_pairwise_corr.csv` / 策略墳場記錄）對照驗證。
> 結論先講：**本系統現在對待技術分析的方式（風控 + 狀態描述 + 橫斷面排序，不做方向預測），
> 就是技術分析在數學上唯一站得住的用法** — 不是妥協，是第一性推導的必然終點。

---

## 第 0 層：先問價格是什麼

價格不是物理量，是**最近一筆成交的共識點** — 某個買方與某個賣方在那一瞬間對價值的交集。所以：

- 價格序列 = 市場參與者**已經處理完的資訊**的聚合軌跡
- 成交量 = 共識形成時的分歧強度

這是唯一的公理。技術分析的一切都建立在這上面。

## 第 1 層：技術指標是什麼 — 數學上全是同一條序列的變換

`technical_analysis.py::calculate_all_indicators()` 計算 MA/BB/BIAS/ATR/Ichimoku/RSI/KD/MACD/OBV/DMI/EFI/TD Sequential。
從訊號處理的角度，它們**沒有一個引入新資訊**：

| 指標 | 本質 |
|---|---|
| MA / EMA | 低通濾波器（uniform / 指數衰減卷積核） |
| MACD | 兩個低通濾波器之差 = 帶通濾波器 |
| RSI / KD / BB %B | 同一個「偏離程度」的不同歸一化 |
| ATR / std20 | 二階矩（波動率）估計量 |
| ADX / DMI | 方向性位移的平滑比率 |
| OBV / EFI | 量價乘積的累積 / 平滑 |
| TD 九轉 | 計數器 |

**資訊論的鐵律（Data Processing Inequality）**：任何對 `(P, V)` 的確定性變換，關於未來的資訊量只會 ≤ 原序列。
一百個指標 = 同一條序列的一百種視角。

本系統自己證明了這點 — `reports/indicator_pairwise_corr.csv`：

```
MA20偏離 vs VWAP偏離   ρ = 0.93
MA20偏離 vs BB %B      ρ = 0.89
MA20偏離 vs RSI偏離    ρ = 0.88
RSI偏離  vs ADX×方向   ρ = 0.83
```

教科書裡「均線派」「擺盪派」「量能派」吵了幾十年的東西，實測共線性 0.8~0.93 —
**它們是同一個訊號穿不同戲服**。這也是 `analysis_engine.py` T1 把 BIAS/VWAP_dev/BB_pct/RSI_dev/EFI
五個合併成單一 Mean Reversion Composite 的原因：它們本來就是一個東西。

## 第 2 層：真正的命題只有一條

把所有指標剝掉之後，技術分析的全部主張濃縮成一個可檢驗的數學命題：

> **E[ r(t+1) | 價格與成交量的歷史 ] ≠ E[ r(t+1) ]**
> （過去的量價序列，對未來報酬有條件預測力）

弱式效率市場假說說「不成立」。但 EMH 不是物理定律，是均衡條件 — Grossman-Stiglitz 悖論：
若市場完全有效，沒人有動機收集資訊，市場就無法有效。所以市場必然「接近但不完全」有效，
殘餘可預測性**可以存在**，但只能來自三種第一性來源：

1. **行為偏誤的系統性殘留** — 反應不足（→ 3-12 個月橫斷面動量）、過度反應（→ 短期與長期反轉）、處分效應
2. **結構性摩擦** — 流動性提供的補償（短期反轉）、強制性交易（融資斷頭、ETF 調倉）
3. **資訊擴散速度差** — 量先價行（知情者交易先反映在量）

注意：這三種來源產生的可預測性全部是**統計性、橫斷面、月級別**的，
沒有一種會產生「KD 黃金交叉買進」這種點對點的方向訊號。

三件第一性層級的事實：

**(a) 教科書的多頭訊號在台股短線是負的。** 「站上均線」「多頭排列」「MACD 翻紅」「Supertrend 翻多」—
全部負 IC。不是零（無資訊），是**負**（反資訊）：在日線尺度，看起來最「多頭」的狀態系統性對應較差的後續報酬。
機制 = 短期反轉 + 散戶在最顯眼的訊號處擁擠追價。

**(b) 偏離類指標 1-5 天強烈均值回歸，到 20 天衰減到零。** 日線技術訊號的資訊半衰期不到一週，
而那一週的方向跟教科書講的相反。

**(c) 唯二正 IC 的，剛好都不是「價的方向」。** RVOL 是**量**（資訊擴散的 footprint），
Squeeze 是**波動率狀態**（壓縮後必有釋放，但不告訴你方向）。
完全符合第一性推導：價格方向的變換沒有殘餘資訊，**量與波動率這兩個「非方向維度」才有**。
（注意：ATR% 的強烈負 IC 看似「低波動異象台股版」，但 2026-06-07 驗證證明那是 **non-tradeable rank 統計量** —
rank-IC 由中位數決定、籃子報酬由右尾決定，equal-weight 低波動籃子 LS spread 16/18 年為負，
做成組合就被高波動肥尾翻號。見 `reports/rvol_atr_factor_validation.md`。）

## 第 4 層：為什麼會這樣 — 四個結構性機制

1. **零壁壘套利** — 指標公開、計算成本為零、人人同時看到。任何曾經存在的 edge 會被最快速度套利掉
   （Adaptive Market Hypothesis；McLean & Pontiff 實測學術異象發表後平均衰減 58%）。
   「夠多人看 60MA 所以它有效」的自我實現論，實際產生的是停損單叢集這種**微觀流動性事件**，
   收割者是造市商，不是看圖的人。本系統數據：MA 排列 20d IC = −0.015，全市場最多人看的訊號，預測力為負。

2. **自由度災難** — 每個指標都有參數（14、20、9、26...），參數組合空間指數成長。
   回測**必然**找得到歷史最優組合 — 那不是發現規律，是 data mining。
   Trend-DMI-SAR 的 26 個變體就是誠實版本的展示：把參數空間掃開，0 個存活。

3. **訊號頻率 × 交易成本** — 點訊號高換手。台股單邊摩擦 ~0.2-0.4%，
   一年翻倉 10 次就要 4-8% 的 gross alpha 才打平 — 而上表最強的訊號 IC 才 0.018。

4. **非平穩性** — 固定參數的指標隱含「價格生成過程平穩」的假設。
   Regime 一換（2020 covid、2024 yen carry、2025 tariff），歷史擬合全部歸零。
   HMM regime 權重做了還是只有 IC +0.016，就是這個原因。

## 第 5 層：技術分析剩下什麼 — 活下來的全部不是「預測」

把第一性原理推到底，技術分析有四個**合法**的剩餘角色，而本系統演化後保留的剛好就是這四個：

| 合法角色 | 本質 | 系統中的對應 |
|---|---|---|
| **狀態描述（儀表板）** | 「現在是什麼狀態」不是「明天往哪走」 | 崩盤預警 S2/S3 的 `ma_dist_60` — single feature 打贏 multinomial/composite ML；ATR_pct 描述波動 regime |
| **風險控制（安全帶）** | 截斷損失分佈的尾部，價值**不依賴**預測力 | `ATR_Stop`（Chandelier Exit）— 不猜方向，保證單筆最大損失有界 |
| **橫斷面排序（排序器）** | 學術動量異象的正確用法：相對強弱 + 月級持有 | QM 的 Trend20/Body30（質量 gate **之後**的動量確認）、強勢股週榜的 52 週新高/13 週累積 |
| **執行錨定** | 進出場分批的參考座標 | 個股分析 tab 的價位區間 |

四者的共同點：**沒有一個回答「方向往哪」**。儀表板描述現在、安全帶控制損失、
排序器是橫斷面不是時間序列、執行錨定是座標不是訊號。

真正的方向性 alpha 在哪？系統的 production 答案：Whale Picks（籌碼結構 + 質量 gate，CAGR 28% / Sharpe 1.67）。
籌碼是**還沒完全反映進價格的資訊**（第 2 層來源 3），質量是基本面的橫斷面差異 — 都不是價格序列的變換。

## 結論

> **技術分析的第一性錯誤，是把「價格的變換」當成「價格之外的資訊」。**

- 指標 = f(P, V)，資料處理不等式保證它不可能比原序列知道更多未來
- 原序列裡殘存的可預測性只有三種來源（行為殘留 / 結構摩擦 / 資訊擴散），
  全是統計性、橫斷面、月級別 — 不存在點訊號形式的方向預測
- 台股全市場實測：教科書多頭訊號 1-20 天 IC 全面為**負**，
  唯二正的（量、波動壓縮）恰好都不是方向指標
- 它的合法身分是**儀表板 + 安全帶 + 排序器**，不是水晶球

本系統五輪驗證的演化路徑 — 技術 timing 全 D → 背離降級為 UI 警語 → alpha 移到籌碼+質量 →
技術指標收斂到風控與狀態描述 — 是第一性原理推導的必然終點。

## 附錄：兩條延伸的驗證結果（2026-06-07 當日驗畢，詳見 `reports/rvol_atr_factor_validation.md`）

1. **log(RVOL) → MARGINAL（informational ranking-tilt only，不進 Whale composite）** —
   全表唯一 rank-IC 與 decile spread 同號、單調（mono ρ +0.99）、流動性過濾後存活、
   walk-forward 18/18 年正（liq_50m 13/18）的訊號；但 edge 太薄：20d 多空 LS spread gross 僅 +0.44~0.74%，
   **成本後在可交易池（liq_50m）歸零/轉負（−0.06~−0.26%）**。只能當 long-only 排序加分（不付兩腿成本），
   不能當獨立多空策略。若要上 production 必須先在 PIT panel（含下市股）重跑確認薄 edge 不消失。
2. **ATR% 反向因子 → FAIL（D 級歸檔）** — 雙重死因：(1) 原 h=60 D1 +5.87%/bot-10 +28% 是
   **stale-price reprice artifact**（根因：panel 12,823 列 Close≤0，近零價股 ATR% 假低），
   清洗 + 流動性過濾後完全消失；(2) 乾淨資料下方向也相反 — 高波動端平均報酬更高
   （右尾 6.7% 股票 20d 漲 >30% 集中在 D10），低波動籃子 LS spread 16/18 年為負。
   「低波動篩選當負向 filter」反而會篩掉較高報酬的股票。ATR% 的合法身分維持第 5 層的
   「波動 regime 描述 + ATR_Stop 風控」，**不是橫斷面選股因子** — 與本文主論點自洽。

> 這兩條驗證本身就是第 3 層論點的再一次演示：**看起來像 alpha 的 IC，攤開到可交易組合層面
> 不是 artifact（ATR%）就是付不起成本的薄 edge（RVOL）**。SOP-10「IC 全綠不能上線」再次應驗。
