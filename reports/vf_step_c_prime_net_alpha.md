## 5. Caveats

- Round-trip cost 0.4% 是 TW 股票中等估算（手續費打折 + 證交稅）；ETF 會更低（~0.15%）
- TWII TR 用 3.5% dividend yield 平均化，實際 2023-2025 台股殖利率偏低（AI 股票殖利率 < 1%）
- Turnover 計算只看 holdings set 變化，未考慮權重 drift 造成的 rebal 成本
- non-volatile 週 Value 側全空手 = 全出場 = cost 算 1.0 turnover 半 round-trip
- 小型股 slippage 未計（可能使實際成本比 0.4% 高 20-30bp）
