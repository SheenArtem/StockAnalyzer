# Whale Picks — 現實執行成本理論稽核

> **這是理論模型 (目前無真實成交資料)。** 所有成本來自 ledger 推算的換手率 +
> 公開費率 + 平方根衝擊模型，非實際成交回報。假設全寫明於下，數字隨假設變動。
>
> 產出: `tools/whale_picks_cost_audit.py` (end-to-end exit 0)
> 資料: `data/whale_picks/trade_ledger.parquet` (833 round-trips, 2016-01 ~ 2026-05) +
> `data_cache/backtest/ohlcv_tw.parquet` (Volume 已確認為**股數**, dollar_TV = Close×Volume)

## 0. 核心結論 (先寫)

| 問題 | 答案 |
|---|---|
| **「-0.5%/年 slippage」站得住嗎?** | **不站得住，嚴重低估。** 連同 backtest 的「1.8% 固定」一起算，假設總成本 ~2.3%/年。**光固定成本 (稅+手續費) 就 3.3%~4.1%/年**，slippage 還沒算。 |
| **真實年成本估多少?** | 個人規模 (1M/檔) **~5.2%/年**；機構規模 (50M/檔) **~16%/年**。固定段 ~3.3-4.1% + 衝擊段 1.8%~12.9% (隨 AUM)。 |
| **誠實 CAGR/Sharpe (4 折手續費)** | 1M: **CAGR 23.3% / Sharpe 1.46**｜5M: 20.5% / 1.31｜20M: 15.7% / 1.03｜50M: 10.3% / 0.71。(gross 基準 29.8% / 1.79) |
| **哪個規模開始被衝擊吃掉?** | **NT$20M/檔 (總 AUM ~236M)** 起衝擊段 >固定段且 p90 部位 >ADV，alpha 轉負 (跑輸 TWII)。**個人規模 (≤5M/檔) 安全。** |

**一句話**: 那句註解既低估「每 round-trip 的固定費率 (用 0.3% 而非實際 0.47-0.585%)」又沒意識到「679% 的年換手」—— 真實固定成本是它的 ~1.8 倍，且 slippage 在中小型部位另外要再吃 1.8%↑；但好消息是**用戶個人規模 (每檔 1-5M) 衝擊很小，誠實 Sharpe 仍有 1.3-1.5，alpha 仍正**。

## 1. 換手率事實 (推翻 flat 0.5%/年 的根因)

backtest 註解假設「~6 round-trips/年 × 0.3% = 1.8% + 0.5% slip」。從 ledger 實算:

| 指標 | 值 | 說明 |
|---|---|---|
| 平均持有期 | **1.77 月** (中位 1 月) | 613/833 筆只持有 1 個月就換掉 |
| 每個 portfolio slot 年 round-trip | **6.79 次/年** | = 12 / 1.77 |
| 平均同時持股 | **11.78 檔** | K=10 目標，多月持有重疊使略高 |
| 全組合年 round-trips | **~80 次/年** | 11.78 slot × 6.79 |
| **年換手率 (one-way)** | **~679%** | 每年資金週轉近 6.8 圈 |

**根因拆解 — backtest 註解錯在哪**:
1. **每 round-trip 費率寫太低**: 用 0.3%，但台股實際 round-trip 固定 = 賣稅 0.3% + 手續費×2 = **0.471% (4折) ~ 0.585% (無折)**。
2. **沒意識到換手是 per-slot 不是 per-portfolio**: 「6 round-trips/年」誤當成整個組合一年只換 6 次；實際是**每個位置**一年換 6.79 次。
3. 兩者疊加: 固定成本 = 6.79 × 0.471% = **3.20%** ~ 6.79 × 0.585% = **3.97%/年** (NAV 灌入精算 3.33%/4.14%)，是假設 1.8% 的 **~1.8-2.2 倍**。
4. **slippage 0.5% 是另一筆**，且只在個人規模成立 (見 §3)。

## 2. 成本模型假設 (寫明)

### 2.1 固定成本 (確定)
- 賣出證交稅 **0.3%** (賣方一律)
- 手續費 **0.1425%/邊** (官方上限) — 兩情境: 無折 0.1425% / **4 折 0.0855%** (現實券商常見)
- round-trip 固定 = 賣稅 + 手續費×2 = **0.471% (4折) / 0.585% (無折)**

### 2.2 市場衝擊 (平方根模型)
```
impact_oneway ≈ c · σ_daily · sqrt(部位名目 / ADV_60d)
c = 0.6                  (學術常用 0.5~1, 取偏保守下緣)
round-trip 衝擊 = 2 × impact_oneway   (進出各一次)
```
- `σ_daily`: entry 前 trailing-60d 日報酬標準差 (中位 **1.49%**)
- `ADV_60d`: entry 前 trailing-60d 平均成交額 TWD (中位 **0.95 億**)
- 部位名目 = AUM_per_stock (= 總 AUM / K)

### 2.3 灌入 NAV 方式
每筆 round-trip 在 ledger 的 entry_date 課 (手續費_buy + impact_buy)×weight，
exit_date 課 (手續費_sell + 證交稅 + impact_sell)×weight，
weight = 1 / n_held(該交易日) (equal-weight，與 baseline backtest 同口徑)。

### 2.4 AUM 情境 (每檔部位名目 → 總 AUM ≈ ×11.8)
| 標籤 | 每檔名目 | ≈ 總 AUM | 對象 |
|---|---|---|---|
| 1M | NT$1M | ~12M | 個人散戶 |
| 5M | NT$5M | ~59M | 大戶/小私募 |
| 20M | NT$20M | ~236M | 中型機構 |
| 50M | NT$50M | ~590M | 大型機構 |

### 2.5 未納入 / 已知限制
- **未計股息**: TWII 與 Whale Picks 兩邊皆未計 (殖利率 ~3-5%/年大致互抵 alpha 比較)。
- **平方根模型在 participation >100% ADV 失效** → 20M/50M 的衝擊段為**下限 (實際更糟)**，因為部位已超過整日成交量，需拆數日成交、承受訊號衰退甚至無法成交 (見 §3 警示)。
- gross 基準用本腳本重算 (CAGR 29.79% / Sharpe 1.793)，與 `portfolio_stats.json` (29.99/1.804) 差 <0.2pp，屬重算誤差。
- 假設**全部位都能在 entry/exit 日以收盤價成交**；冷門股實務上未必有對手盤。
- 未計借券/當沖等費用 (本策略為純多現股，不適用)。

## 3. 流動性與 participation (中小型關鍵)

ledger 827 個有效部位的 entry-前 ADV_60d:

| 百分位 | ADV (億 TWD) |
|---|---|
| p05 | 0.116 |
| p10 | 0.137 |
| p25 | 0.233 |
| **p50 (中位)** | **0.951** |
| p75 | 2.59 |
| p90 | 6.33 |
| p95 | 10.9 |

50% 的部位 ADV <1 億，最薄 10% <0.137 億 — 確實是中小型為主、左尾有極薄名單。

**participation (部位/ADV) 與單邊衝擊 by AUM**:

| AUM/檔 | 中位 participation | p90 | 最薄 | 單邊衝擊中位 | 單邊衝擊 p90 | 單邊衝擊最大 |
|---|---|---|---|---|---|---|
| **1M** | 1.05% | 7.3% | 13.8% | 9.3 bps | 27 bps | 82 bps |
| 5M | 5.3% | 36.6% | 69% | 20.9 bps | 61 bps | 184 bps |
| 20M | 21.0% | **146%** ⚠ | 276% | 41.8 bps | 121 bps | 367 bps |
| 50M | 52.6% | **366%** ⚠ | 691% | 66.0 bps | 191 bps | 581 bps |

⚠ **20M 以上 p90 participation 已 >100% ADV** = 單一部位要吃掉整日成交量的 1.5-3.7 倍。
此時平方根模型**已不成立**，衝擊成本被**低估**；現實中這些薄名單根本無法在 rebalance 日完成
進出，或需拆數日承受訊號衰退。所以 20M/50M 的「誠實 CAGR」其實是**過於樂觀的上界**。

### 對照 backtest 假設
- backtest 假設總成本 = 1.8% + 0.5% = **2.3%/年** → 隱含淨 CAGR ~27.5%。
- 即使**最樂觀**情境 (個人 1M + 4折): 真實總成本 **5.15%/年**，是假設的 **2.2 倍**；淨 CAGR 23.3% (比假設的 27.5% 再低 4.2pp)。
- backtest 那句註解對**任何 AUM 都偏樂觀**，最小低估 2.2x，機構規模低估 7x。

## 5. 逐題回答

**(1) -0.5%/年 假設站得住嗎? 實際年成本?**
不站得住。連 backtest 自己加的「1.8% 固定」一起看 (合計假設 2.3%/年) 都嚴重低估。
- **固定成本 (稅+手續費) 就 3.3-4.1%/年** (679% 年換手 × 0.47-0.585% round-trip)，已超過全部假設。
- slippage/衝擊另計: 個人規模 +1.8%/年、機構規模 +4~13%/年。
- **真實總成本: 個人 ~5.2%/年, 機構 ~16%/年。**

**(2) 各 AUM 誠實 CAGR/Sharpe (4折手續費)**
- 1M/檔 (~12M 總): **CAGR 23.3% / Sharpe 1.46** — 仍是好策略
- 5M/檔 (~59M 總): CAGR 20.5% / Sharpe 1.31
- 20M/檔 (~236M 總): CAGR 15.7% / Sharpe 1.03 — alpha 轉負
- 50M/檔 (~590M 總): CAGR 10.3% / Sharpe 0.71 — 跑輸 TWII，且此值還是高估

**(3) 哪個規模被衝擊吃掉? 個人安全嗎?**
- **NT$20M/檔 (總 ~236M) 是臨界點**: 衝擊成本 (8.1%) > 固定成本 (3.3%)，p90 部位 >整日 ADV，
  vs TWII alpha 轉負。再大 (50M) 模型本身失效，實務無法執行。
- **個人規模 (≤5M/檔, 總 ≤60M) 安全**: 衝擊段 ≤4%，誠實 Sharpe ≥1.3，alpha 仍正 (+3~6pp)。
  最薄名單在 1M/檔下最差也只吃整日量的 14%、單邊衝擊 82bps，可接受。

**(4) 一句結論**
那句 -0.5%/年 (及合計 2.3%) 對任何規模都太樂觀 (真實固定成本就 3.3-4.1%，因 679% 年換手被忽略)，
但用戶個人規模下誠實 CAGR 仍有 23%、Sharpe 1.46、alpha +6pp，**策略對散戶站得住，只是別期待 backtest 的 30%**；
**一旦資金放大到每檔 20M (總 2 億) 以上，高換手 + 中小型流動性會把 alpha 吃光，此策略不可規模化到機構等級。**
