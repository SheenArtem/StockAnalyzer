### 名詞說明

- **trig_rate**：forward 20d 內 SL 被收盤價打穿的比例
- **false_stop_rate**：SL 觸發後 20d 內最高收盤 >= entry 的比例（被噪音打穿但股票其實該續漲）
- **realized_mean**：實際策略報酬 = 若觸發 SL 取觸發日報酬，未觸發持有至 20d
- **realized_sharpe**：realized_mean / realized_std（單筆 20d 樣本 Sharpe，非年化）
- **win_rate**：realized > 0 的比例
- **trig_ret_mean**：僅觸發樣本的平均報酬（負值正常）
- **rr (R:R)**：20d 最大漲幅 / (entry - SL)，>5 代表期望值虛高（通常因 SL 太近）

## 5. 決策規則 & 建議

### 判斷規則

- **A 級**（採納）：高 ATR 假停損降幅 ≥ 20% 或 全樣本期望值差 ≥ 0.5%
- **B 級**（採納，中等信心）：降幅 ≥ 10% 且 期望值差 ≥ 0.2% 且 勝率差 ≥ 5pp
- **C 級**（邊緣）：降幅 ≥ 10% 或 期望值差 ≥ 0.2%
- **D 級**（棄用）：以上皆否

### 實測結果

| 指標 | 數值 |
|---|---|
| 高 ATR 假停損降幅 | 10.1% (邊緣 ≥10%) |
| 全樣本期望值差 (B-A) | +0.33% (邊緣 ≥0.2%) |
| 勝率差 (B-A) | +8.3pp (達標 ≥5pp) |
| **綜合判定** | **B 級** |

### 建議修改 `exit_manager.py`

```diff
-DEFAULT_MIN_SL_GAP = 0.03         # 停損距進場至少 3%
+DEFAULT_MIN_SL_GAP = 0.03         # 停損距進場至少 3%（固定下限）
+
+def compute_min_sl_gap(atr_pct=None):
+    """動態 min_sl_gap：至少 3%，且至少 1.5 個日 ATR。"""
+    if atr_pct is None or atr_pct <= 0:
+        return DEFAULT_MIN_SL_GAP
+    return max(DEFAULT_MIN_SL_GAP, atr_pct * 1.5 / 100.0)
```

在 `compute_exit_plan()` 中：
```python
min_sl_gap = compute_min_sl_gap(atr_pct) if atr_pct is not None else min_sl_gap
```
