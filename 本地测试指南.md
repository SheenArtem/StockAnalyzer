# 📋 StockAnalyzer 本地测试指南

## 🚀 快速开始

### 方法一：使用批次文件（最简单）

直接双击运行：
```
run_app.bat
```

这个脚本会自动：
1. 安装所有依赖包
2. 启动 Streamlit 应用
3. 自动在浏览器打开

---

### 方法二：命令行运行（推荐）

#### 1️⃣ 安装依赖

首先确保已安装 Python（建议 3.9+）

```powershell
# 在项目目录下打开 PowerShell 或 CMD
cd C:\GIT\StockAnalyzer

# 安装依赖包
pip install -r requirements.txt
```

#### 2️⃣ 启动应用

```powershell
streamlit run app.py
```

#### 3️⃣ 访问应用

浏览器会自动打开，如果没有，手动访问：
```
http://localhost:8501
```

✅ **数据单位正确**

- 当日总量：应该是合理的万位数（例如：34,688 张）
- ❌ 不应该看到百万级的异常大数字

### 3. 法人买卖超动态门槛

✅ **智能判断**
- 高价股：考虑资金金额（例如：300百万）
- 低价股：考虑成交量占比（例如：占均量18.5%）
- 显示详细说明文字

## 🛠️ 常见问题排除

### 问题1：端口被占用

```
Error: Port 8501 is already in use
```

**解决方案：**
```powershell
# 方法A：更换端口
streamlit run app.py --server.port 8502

# 方法B：找出并关闭占用进程
netstat -ano | findstr :8501
taskkill /PID <进程ID> /F
```

### 问题2：依赖包安装失败

```
ERROR: Could not find a version that satisfies the requirement...
```

**解决方案：**
```powershell
# 升级 pip
python -m pip install --upgrade pip

# 逐个安装核心包
pip install streamlit pandas yfinance finmind plotly
```

### 问题3：数据加载慢

**原因：** 首次运行需要从 API 下载数据

**解决方案：**
- 第一次会比较慢（约30秒）
- 数据会缓存在 `data_cache/` 目录
- 后续加载会很快（1-2秒）

### 问题4：找不到模块

```
ModuleNotFoundError: No module named 'xxx'
```

**解决方案：**
```powershell
# 确保在项目目录下
cd C:\GIT\StockAnalyzer

# 重新安装依赖
pip install -r requirements.txt --force-reinstall
```

## 📝 性能优化建议

### 加快启动速度

1. **使用虚拟环境**
```powershell

python -m venv venv

# 激活虚拟环境
.\venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

2. **清理缓存**
```powershell
# Streamlit 缓存
streamlit cache clear

# 或删除缓存文件夹
rmdir /s /q .streamlit\cache
```
